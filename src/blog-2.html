Find the Largest Product of Any Three Numbers

May 12, 2020 by Andrew Doolittle

Write a function that finds the largest possible product of any three numbers. The three numbers may be negative

Let's lay out the important details.

Input: An array of integers.

Output: largest product of the three integers.

Edge cases: Negative numbers, an array length of less than 3.

Now let's create a few examples to help us visualize some patterns.

Example 1

Input: [1,5,8,3]

In this example, we know that getting the product using the highest numbers will return the correct output. There’s no point of using 1 because we know any number multiplied by 1 returns itself.

9 * 5 * 3 = 135

Example 2

Input: [100, 91, -3, 40]

Similar to the first example, we don’t want to multiply by a negative as that results in a negative number.

100 * 91 * 40 = 364000

Example 3

Input: [-4, -10, 2, 3, 1]

This one is tricky because we finally have a situation where there are two negative numbers and we know two negative numbers return a positive. One or three cases of negative numbers will return a negative product. This means we either use two negatives or none at all. Let’s test some combinations to find the largest product.

3 * 2 * -4 = -24

3 * 2 * -10 = -60

3 * 2 * 1 = 6

-10 * - 4 * 2 = 80

-10 * -4 * 3 = 120

Example 4

Input: [-9, 3, 7, -2, 10, -11]

After these four examples, we can see a pattern in how to find the highest product. First, we have to sort our array in a descending order. We then find the product of the first three elements.

However, because there’s a chance for a negative numbers to be present, we should also compare the product of the last two numbers and only two because we need two negatives to return a positive. Then we multiply the last two elements with the first element. Once we have two products, we can compare whichever is higher and return that.

10 * 7 * 3 = 210

-11 * -9 * 10 = 990

Solution in JavaScript
                         
                             var largestProductOfThree = function(array) {
                                // TODO: everything
                                //ensure input is array
                                if (!Array.isArray(array)) {
                                    return null;
                                }

                                //ensure array is proper length
                                if (array.length < 3) {
                                    return null;
                                }

                                //create  sorted copy to not mutate
                                let sortedAscend = array.slice().sort(function(a,b) {return a - b})

                                //get length
                                let len = sortedAscend.length;

                                //track using 2 products variables
                                //A product of last 3 (highest positive numbers)
                                let productA = sortedAscend[len-1] * sortedAscend[len-2] * sortedAscend[len-3];

                                //B product of first 2 lowest and last highest (neg * neg * pos = pos)
                                let productB = sortedAscend[0] * sortedAscend[1] * sortedAscend[len-1];

                                return Math.max(productA, productB);
                            };
